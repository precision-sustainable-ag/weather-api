This API provides NLDAS-2 forcing data and MRMS precipitation data on an hourly or daily basis.

It also provides miscellaneous other location-based data, including elevation, watersheds, hardiness zones, MLRA, etc.

  • NLDAS-2: 0.125° × 0.125° resolution, available from 2015-01-01 to within the last 3 days (for quality-control latency).
  • MRMS: 1 km × 1 km resolution, available from 2015-06-05 to within the last 2 hours.
