This API provides several categories of gridded weather datasets:

  1. NLDAS-2 Forcing Data
    • Resolution: 0.125° × 0.125° (~14 km × 14 km)
    • Coverage: CONUS, 25°N–53°N, 125°W–67°W
    • Availability: 2015-01-01 to present (with ~3-day latency)
    • Content: Hourly atmospheric forcing variables used to drive the land-surface models (temperature, humidity, wind, radiation, precipitation, etc.).
  2. NLDAS-2 Mosaic Land-Surface Model (LSM)
    • Resolution: 0.125° × 0.125° (identical to the forcing grid)
    • Availability: 2015-01-01 to present
    • Content: Mosaic LSM surface and subsurface diagnostics, including energy fluxes, soil moisture/temperature layers, snow fields, canopy variables, etc.
  3. MRMS Precipitation
    • Resolution: 1 km × 1 km
    • Availability: 2015-06-05 to present (within ~2 hours)
    • Content: Radar-based precipitation totals (hourly).

The API also provides miscellaneous other location-based data, including elevation, watersheds, hardiness zones, MLRA, etc.

NLDAS-2 Mosaic Land-Surface Model (LSM) Parameters

These variables describe surface energy balance, vegetation, soil layers, canopy water, snowpack, and hydrology, and are available through the attributes parameter.

Parameter Units Description
